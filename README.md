## 💡 Project Overview
Punctuality is the lifeblood of mass transit. This project transforms raw, fragmented quarterly reports from the **Amtrak Intercity Passenger Rail Service** into a cohesive, geospatially-augmented analytical engine. 

By combining rigorous data engineering principles with advanced statistical analysis, this repository uncovers actionable insights into station efficiency, route performance, and punctuality—with a specialized deep dive into the **State-Supported Michigan Service** (*Blue Water, Pere Marquette, Wolverine*).

---

## 🎯 Key Business Insights
Through extensive data exploration, this project uncovered several critical operational patterns:
* **Geospatial Delay Tracking:** Identified specific stations along the Michigan service lines that consistently act as bottlenecks, causing cascading delays across the network.
* **Predictive Benchmarking:** Evaluated actual field delay times against predictions generated by a **Simple Least-Squares Linear Regression** model based on route distance (*route miles*).
* **Directional Variance:** Uncovered significant statistical differences in "average minutes late" when comparing Eastbound vs. Westbound trains on the exact same routes.
